# Express-Biztime

## Seed test db
    - change to biztime_test in data.sql
    - psql biztime_test
    - \i data.sql

## Seed Production db
    - change to biztime in data.sql
    - psql biztime
    - \i data.sql

## Run Test
    - npm run test

## Run app in normal mode
    - npm run start
  
  
## Run app in debug mode
    - npm run dev
